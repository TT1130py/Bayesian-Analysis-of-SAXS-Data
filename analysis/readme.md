## List of content

- **compare_weights.py:** compare one dataset of weighted structures with another and find average weight for each structure
- **cumulative_weights.py:** calculate a cumulative density function for both prior and posterior weights of one dataset
- **curve_compare.py:** compare any number of posterior SAXS curves from any grid point against the experimental SAXS curve and calculate the approximate radius of gyration
- **ensemble_fit.py:** investigate the prior and posterior SAXS curves of one dataset against the experimental SAXS
- **file_curve.py:** examine the calculated SAXS curves of any individual pdb file against the experimental SAXS
- **plot_theta.py:** plot an L curve for a theta scan run (theta_scan.py)
- **Rg_compare.py:** investigate the prior and posterior radii of gyration probability density shifts against the experimental Rg
- **Rg_gps.py:** compare any number of posterior Rg probability distributions from any grid point against the experimental Rg
- **theta_curve_fit.py:** compare best fit ensemble averaged SAXS curves across different iBME runs (of different theta values) against the experimental SAXS
- **theta_scan.py:** perform multiple iBME runs on the same grid simulation across a defined range of theta values
